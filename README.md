# Sticky Notes Django Application

A beautiful, full-featured sticky notes web application built with Django. Create, organize, and manage your notes with a clean, responsive interface.


## Features

- **Full CRUD Operations**: Create, Read, Update, and Delete sticky notes
- **Beautiful UI**: Sticky note-inspired design with gradient colors and animations
- **Responsive Design**: Works perfectly on desktop, tablet, and mobile
- **Django Admin Interface**: Built-in admin panel for note management
- **Comprehensive Testing**: Unit tests covering models, views, forms, and URLs
- **Professional Architecture**: Follows Django best practices and MVT pattern

## Technologies Used

- **Backend**: Django 6.0
- **Database**: SQLite (development), ready for MariaDB/PostgreSQL
- **Frontend**: HTML5, CSS3 with custom styling
- **Testing**: Django Test Framework
- **Version Control**: Git & GitHub
